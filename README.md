# Fling Arrows

A puzzle game where you slide arrow blocks off the board in the right order.

**50 levels** across 5 difficulty tiers (Easy, Medium, Hard, Expert, Master) with shaped grids â€” diamonds, crosses, hearts, stars, and more.

## Play

[Play in your browser](https://dinislam-y.github.io/fling-arrows-web/)

## How to play

- Tap an arrow block to slide it in its direction
- Blocks slide off the board if the path is clear
- If the path is blocked, you lose a life
- Clear all blocks to win the level
- 3 lives per level, no timer

## Tech

Built with Flutter (Dart), pure CustomPainter rendering (no game engine).
